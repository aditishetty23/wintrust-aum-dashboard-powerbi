# Wintrust AUM Dashboard (Power BI)

This project presents a financial analytics dashboard built in Power BI to analyze advisor performance and Assets Under Management (AUM) growth. The dashboard provides interactive KPIs, time-series trends, and product-level breakdowns to identify top advisors and at-risk households.

## Dataset

The dataset represents a mock wealth management portfolio used to analyze advisor performance and Assets Under Management (AUM).

### Key Fields
- **SnapshotDate** – Monthly snapshot of portfolio values
- **AdvisorID / AdvisorName** – Financial advisor managing the accounts
- **Region** – Geographic region of the advisor
- **AccountID** – Unique investment account identifier
- **ClientName** – Client associated with the account
- **HouseholdID** – Household grouping for related accounts
- **ProductType** – Investment category (Equity, ETF, Mutual Fund, Bonds, Cash)
- **AUM** – Assets Under Management for the account
- **Inflows** – New investments added during the period
- **Outflows** – Withdrawals during the period

The dataset enables analysis of portfolio growth, advisor performance, and asset allocation across different product types.
## Key Features
- Total Assets Under Management (AUM) analysis
- Net New Assets tracking
- Advisor performance comparison
- Product type allocation (Equity, ETF, Mutual Fund, Bonds, Cash)
- Household segmentation (Active vs At-Risk)
- Time-based filtering (Year, Quarter, Month, Day)

## Dashboard Insights
- Identify top-performing financial advisors based on AUM contribution
- Track growth trends in assets under management
- Analyze distribution of investment products across portfolios
- Detect households that may require retention strategies

## Tools & Technologies
- Power BI
- Data Modeling
- DAX Measures
- Financial Data Visualization
